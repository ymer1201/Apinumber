API Documentation

Overview

This API allows you to validate and retrieve information about phone numbers. It provides details such as country code, carrier, and line type.

Endpoints

Validate Phone Number

URL: https://api.apilayer.com/number_verification/validate?number={phone_number}

Method: GET

Parameters:

  • number (required): The phone number to validate.

Response Example

    {
        "valid": true,
        "number": "+1234567890",
        "local_format": "1234567890",
        "international_format": "+1234567890",
        "country_prefix": "+1",
        "country_code": "US",
        "country_name": "United States",
        "location": "New York",
        "carrier": "AT&T",
        "line_type": "mobile"
    }

Usage

To use this API, you need to obtain an API key from apilayer.com. Include the API key in the request headers as follows:

    headers = {
        "apikey": "YOUR_API_KEY"
    }
